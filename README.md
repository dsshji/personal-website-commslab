# personal-website-commslab

A portfolio website showcasing four projects made during Communications Lab (Spring 2026) at NYU Abu Dhabi.

**[→ View Live](https://dsshji.github.io/personal-website-commslab/)**

---

## About

Built with a retro y2k/PSP aesthetic — dark palette, pixel fonts, and interactive 3D models as project thumbnails. Each project page features a Three.js 3D object that spins and floats, matching the medium of the work it represents.

## Projects

| | Title | Medium |
|---|---|---|
| 01 | **Stuck in the Loop** | Short film + website |
| 02 | **An Evening of Coming Home** | Interactive digital comic |
| 03 | **The Homecoming** | Audio story website |
| 04 | **comms_lab_exam** | Choose-your-own-adventure video game |

## Features

- 3D models rendered with Three.js (GLB format) — one per project, matching its medium
- Looping background soundtrack with mute/unmute toggle, persistent across pages
- Pixel-style UI with Press Start 2P and Pixelify Sans fonts
- Mouse cursor trail effect
- Clickable 3D models that open the project

## Stack

Plain HTML, CSS, JavaScript — no frameworks. Three.js via CDN importmap. Hosted on GitHub Pages.
